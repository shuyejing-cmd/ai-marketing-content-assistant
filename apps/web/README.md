# AI Marketing Mobile H5

这是本地手机版网页 MVP。当前阶段不接真实 AI、不需要登录、不需要服务器费用，先验证“手机上能不能顺畅生成图文营销包”的流程。

## 本地启动

在项目根目录进入前端应用：

```powershell
cd apps/web
npm.cmd install
npm.cmd run dev
```

电脑浏览器打开：

```text
http://localhost:3000
```

## 手机访问

手机和电脑需要连接同一个 Wi-Fi。

在 Windows PowerShell 查看电脑局域网 IP：

```powershell
ipconfig
```

找到无线网卡里的 `IPv4 地址`，例如：

```text
192.168.1.23
```

手机浏览器打开：

```text
http://192.168.1.23:3000
```

如果手机打不开，优先检查：

- 电脑和手机是否在同一个 Wi-Fi。
- 电脑防火墙是否允许 Node.js 访问专用网络。
- 本地服务是否还在运行。

## 当前能力

- 首页三入口：文案、图片、视频。
- 图片页底部输入框：像聊天一样输入一句话需求。
- 快捷按钮：上传图片、发布渠道、营销场景、风格模板、活动信息。
- 商品图可选：上传后会在 mock 海报里保留商品图；不上传也能生成通用营销图。
- Mock 图文营销包：生成多套标题、发布文案、海报预览。
- 操作按钮：复制文案、下载图片、重新生成、二次修改。
- 本地历史：最近生成记录保存在当前浏览器里。

## 当前不做

- 不接真实 AI 模型。
- 不做登录、积分、支付、订单。
- 不直接发布到微信、小红书、抖音、美团或大众点评。
- 不做复杂手动海报编辑器。
- 不做真实视频生成。
